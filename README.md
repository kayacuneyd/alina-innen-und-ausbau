# ALINA Innen- & Ausbau

Bu proje, ALINA Innen- & Ausbau için oluşturulmuş tek sayfalık modern bir web sitesidir. React ve TypeScript kullanılarak geliştirilmiştir.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
3. Open your browser at `http://localhost:3000`
